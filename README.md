# pyrycode Homebrew Tap

Homebrew formulas for [pyrycode](https://github.com/pyrycode/pyrycode) projects.

## Install

```bash
brew install pyrycode/tap/pyry
```

This is equivalent to:

```bash
brew tap pyrycode/tap
brew install pyry
```

## Formulas

| Formula | Description |
|---|---|
| `pyry` | Process supervisor for Claude Code. See [pyrycode/pyrycode](https://github.com/pyrycode/pyrycode). |

## How it's maintained

Formulas in this tap are auto-generated by [GoReleaser](https://goreleaser.com) on every tagged release of the upstream project. Don't edit formula files by hand — they'll be overwritten on the next release.
